# CricBox 🏏

CricBox is a full-stack web application that allows users to explore and book cricket turfs online.
It provides an easy interface for users to view available turfs and for turf owners to manage their turfs and bookings through a dashboard.

## ✨ Features

* User authentication (Login / Register)
* Browse available cricket turfs
* Turf booking system
* Owner dashboard
* Add, edit and delete turfs
* Booking management
* Responsive user interface
